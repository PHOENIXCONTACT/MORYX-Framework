# Test Case 5 - Delete a Role
## Expected preconditions

Make sure you are logged in using an administrator account and navigated to the roles page. There should also be an existing role in addition to the SuperAdmin role (as is the result of [test case 4](04_add_role.md)).

## Steps

| Step/Instruction | Expected Result | Comment |
|------------------|-----------------|---------|
| Click on the Action dropdown menu and select the Delete button of the role in the table | The role should disapper | The SuperAdmin role should not have this option available |
