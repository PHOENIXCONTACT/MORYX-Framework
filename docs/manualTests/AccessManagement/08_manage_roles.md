# Test Case 8 - Manage Roles

## Expected preconditions

Make sure you are logged in using an administrator account. There should also be an existing role in addition to the SuperAdmin role (as is the result of [test case 4](04_add_role.md)).

## Steps

| Step/Instruction | Expected Result | Comment |
| ---------------- | --------------- | ------- |
| Click on the Action dropdown menu and select the Manage Roles button of the user in the table | You should be navigated to a new page to manage the roles of the user | |
| Select the role in the list and click the Save button | An indicator should be displayed to notify you of a successful update of the role. ||
| Deselect the role in the list and click the Save button | An indicator should be displayed to notify you of a successful update of the role. ||
