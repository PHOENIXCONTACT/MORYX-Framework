# Test Case 7 - Edit a User

## Expected preconditions

Make sure you are logged in using an administrator account.

## Steps

| Step/Instruction | Expected Result | Comment |
| ---------------- | --------------- | ------- |
| Click on the Action dropdown menu and select the Edit button of the user in the table | You should be navigated to a new page to edit the user | |
| Fill in a First Name as well as a Last Name and click on the Save button | You should be navigated back to the users page. Editing the user again should show that the respective fields are prefilled with the values provided before. ||
