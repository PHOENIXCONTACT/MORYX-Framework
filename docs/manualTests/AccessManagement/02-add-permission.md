# Test Case 2 - Add permissions

## Expected preconditions

Make sure you are logged in using an administrator account.

## Steps

| Step/Instruction | Expected Result | Comment |
| ---------------- | --------------- | ------- |
| Click on Permissions | You are navigated to the Permissions page ||
| Fill in the name of a new permission in the text field and click on the add button | The new permission appears in the table below the text field |
| Fill in the same name again and click on the add button | An indicator appears that the permission already exists and the table stays unchanged. ||
