# Test Case 4 - Add a Role

## Expected preconditions

Make sure you are logged in using an administrator account.

## Steps

| Step/Instruction | Expected Result | Comment |
| ---------------- | --------------- | ------- |
| Click on Roles | You are navigated to the Roles page ||
| Fill in the name of a new role in the text field and click on the add button | The new role appears in the table below the text field |
| Fill in the same name again and click on the add button | An indicator appears that the role already exists and the table stays unchanged. ||
