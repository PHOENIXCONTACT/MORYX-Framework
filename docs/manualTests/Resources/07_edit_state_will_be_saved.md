## Expected preconditions
A resource with properties exists

| Step/Instruction | Expected Result | Comment |
|------------------|-----------------|---------|
| Click on the **Resources** button at the shell| The UI switches to the resources page. ||
| Select the created resource | The editing button becomes available ||
