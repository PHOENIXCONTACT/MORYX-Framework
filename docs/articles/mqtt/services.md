# [ResourceEventService](https://github.com/PHOENIXCONTACT/MORYX-Framework/blob/dev/docs/articles/mqtt/resource-event-service.md) 
    Sends messages to the broker when a public event or event exposed using the `ResourceAvailableAsAttribute` is raised.
    When an event is raised the payload sent to the broker is of type `ResourceEventPayload`

## [ResourceSynchronizationService](https://github.com/PHOENIXCONTACT/MORYX-Framework/blob/dev/docs/articles/mqtt/resource-synchronization-service.md)
    Sends messages to the broker when the resource is modified/updated.
